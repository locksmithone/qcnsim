Description
This class implements a Normal distribution traffic generator. Events generated from this class will be distributed in time according to a Normal probability distribution: more information about the Normal distribution at http://en.wikipedia.org/wiki/Normal_distribution

double NormalTrafficGenerator::generateNormalVariate ( )
private

Generate a Normal distribution variate based on member variables mean and standardDeviation.

The Normal variate is generated using C++11 Random library and uses the random number generator obtained from SimulatorGlobals, such that its global seed can be utilized.

Returns
Normal distribution variate based on member variables mean and standardDeviation.

Description
Names for events that will be handled by the Simulator. The events are typically handled in a switch-case strucutre in a main loop.

Arrival of instance of traffic generator.

SET_LINK_DOWN 

Sets link down.

REROUTE_QCN_TRAFFIC 

Reroutes traffic from QCN sensors.

END_PROPAGATION_AT_LINK 

Ends propagation of a PDU in a link. Schedule next event (typically ARRIVAL_AT_NODE).

END_SIMULATION 

End of simulation event. Should be the last event to occur in the simulation, and the Event Chain should have at least this event for soundness.

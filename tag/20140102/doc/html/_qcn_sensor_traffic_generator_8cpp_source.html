220  // Creates a PDU and assigns it class member values and a unique ID obtained from simulatorGlobals.
221  // Notice that the base class member function "createInstance..." is used here instead of the derived override:
222  // the derived override schedules an event, and that is undesired here, since an event can only be scheduled "after" the PDU is created.

SimulatorGlobals::simulationAbsoluteStartTime
double simulationAbsoluteStartTime
Indicates the absolute time measurements should consider as the start of the simulation. This allows a simulation to have a &quot;warm-up&quot; time, which statistics will disregard. To use, just set this variable to the time after which statistics should be calculated, e.g., if set to 5.0, busy times and utilization measurements before 5.0 will be disregarded (by doing the subtraction currentAbsoluteTime - measurementsAbsoluteStartTime).

Description
This class implements a two-parameter Weibull traffic generator. Events generated from this class will be distributed in time according to a Weibull probability distribution: more information about the Weibull distribution at http://en.wikipedia.org/wiki/Weibull_distribution.

double WeibullTrafficGenerator::generateWeibullVariate ( )
private

Generate a Weibull distribution variate based on member variables scale and shape.

The Weibull variate is generated using C++11 Random library and uses the random number generator obtained from SimulatorGlobals, such that its global seed can be utilized.

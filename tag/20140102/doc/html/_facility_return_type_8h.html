Description
Types of return from certain functions within Facility class.
Enumerator
TOKEN_PUT_IN_SERVICE 

Token was successfuly put in service by facility.

FACILITY_BUSY_TOKEN_ENQUEUED 

Facility was busy; token was enqueued.

FACILITY_DOWN_TOKEN_DROPPED 

Facility is down. Token was dropped (or enqueued, depending on the implementation of function Facility::request).

TOKEN_RELEASED_DEQUEUED_SERVICE_EVENT_SCHEDULED 

Token was released from service; token queue was not empty, token at head of queue was scheduled for service at this facility.

TOKEN_RELEASED_QUEUE_EMPTY 

Token was released from service. No token in queue for servicing.

TOKEN_NOT_FOUND 

Token was not found (typically for release) in this facility (service or queue). This is an error condition.

FACILITY_BUSY_QUEUE_FULL_TOKEN_DROPPED 

The queue is full (has reached the imposed limit); token for which service was requested was dropped.

NOT_IMPLEMENTED 

This function has not been implemented yet.

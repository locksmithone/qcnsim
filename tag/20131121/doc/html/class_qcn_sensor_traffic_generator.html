Description
This class implements a Quake-Catcher Network Sensor Traffic Generator. Basically, a simulated sensor will be triggered by a seismic event, and then the sensor will produce a PDU to be sent to a BOINC server. This PDU may contain information such as time of observed event, magnitude, and other sensor-related information.*

Description
The Event class implements an Event for the simulator. An Event has a latency field, or the amount of time it will take for the event to initiate from the current time or clock time; a field for the type of the event; and a generic object associated with the event.

Comparator !=, non-member.

Operator != (not equal) overloading.

This is the overloading of the comparison != operator for use with Event. The result should be FALSE if the left Event has the same contents as the right Event.

Note that this function is simple the negation of == operator. It is being implemented for the sake of fundamental rule of operator overloading.

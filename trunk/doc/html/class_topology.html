Description
This Topology class implements data structures and functions to facilitate management of the network topology part of a QCN simulation.

In particular, this class can concentrate vectors of Nodes, Links, traffic generators, etc. A few utility functions might include finding a link given nodeA and nodeB and getters and setters.

This is a template for future implementations. Use with wisdom. :)

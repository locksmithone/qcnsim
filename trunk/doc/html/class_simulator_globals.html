Detailed Description

Simulator-scope variables and parameters.

Description
Simulator-scope variables and parameters will be here. Most important one is currentAbsoluteTime, which indicates the current, clock time of the simulation. It also includes resources to automatically generate unique token IDs and maintain a simulator-unique random number generator engine and seed.

double SimulatorGlobals::getSimulationDuration ( ) const

Returns the simulation duration in time.

The simulation duration is the current absolute time minus the simulation absolute start time. If the simulation start time is different than zero (when the simulation typically starts), then the time elapsed between zero and simulation duration is considered the "warm-time" period and will be discarded when computing certain statistics, such as facility utilization.

Returns
Simulation duration.

Sets seed for the random number generator engine and seeds the random number generator.

The purpose of this function is basically allow for setting a fixed seed for the random number generator, such that a deterministic pseudorandom sequence is established. This might be useful for debugging purposes, or to reproduce simulator runs from the same initial parameters.
